# 算法与数据结构

## 简介

本仓库收集用 Typescript 实现的算法与数据结构，帮助算法初学者以直观的方式了解、学习算法。

## 数据结构

- 树（Tree）
  - 二叉树（Binary Tree）
  - 二叉搜索树（Binary Search Tree）

## 算法

- 排序（sort）
  - [冒泡排序（bubble sorting）](src/algorithms/sort/bubble-sort/README.md)
  - 插入排序（insertion sorting）
